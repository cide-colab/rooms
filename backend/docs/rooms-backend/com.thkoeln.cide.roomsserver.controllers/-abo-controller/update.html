update

@PutMapping(["{id}"]) fun update(@PathVariable id: String, @RequestBody protocol: AboProtocol): Unit

This method is called to update an abo.

Method is called if the application receives a PUT request to /abos/{id}

Parameters

id - ID of Abo, wich should be updated. This parameter is automatically fetched from request url.

protocol - AboProtocol, Protocol of Abo, wich should be updated with all changes. This parameter is automatically fetched from incoming request body.
