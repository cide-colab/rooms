com.thkoeln.cide.roomsserver.controllers.Controller

A controller represents the lowest level of the backend. An incoming request is forwarded directly to the controller. This is where the mapping of the residual resource to a controller method takes place. Its task is to prepare incoming requests for further processing and prepare the response for the residual response. Access control at resource level also takes place in this layer.
